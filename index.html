Publications

SMMap: Soil Moisture Content Mapping/Estimation Using Synthetic Aperture Radar (SAR) Data

Published in ICSSIT, 2023

This research introduces a novel approach to estimate soil moisture content using Synthetic Aperture Radar (SAR) data and machine learning. Key points include:

  • Utilizes a Regression Convolutional Neural Network (RCNN) to process SAR data from Sentinel-1 C-band radar.
  • Enables soil moisture estimation over large and remote areas without requiring prior knowledge of soil properties.
  • Offers a more efficient alternative to traditional, time-consuming moisture estimation methods.
  • Provides a user-friendly GUI for uploading VV and VH images to predict soil moisture content.

Automatic Certificate Verification using Computer Vision

Published in IDCIoT, 2023

This paper presents an innovative solution to combat fake degrees and certificates using computer vision techniques. The system:

  • Automates the certificate verification process, reducing delays and complexity in handling large volumes.
  • Uses image processing to detect and extract key information from certificates.
  • Employs the EasyOCR module for text recognition from extracted image sections.
  • Implements a client-server architecture for efficient verification against institute databases.
  • Aims to improve organizational growth by ensuring a properly educated and trained workforce.
